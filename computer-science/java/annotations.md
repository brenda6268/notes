Java 的 Annotation 和 Python 的 Decorator 语法非常类似，但是功能和实现完全不同。不过在实际使用过程用，有时候两者还会起到类似的作用，有点像是鲸鱼和鲨鱼，虽然看起来都是鱼，但是一个是哺乳动物，一个是鱼类，只不过是趋同进化。
