# 内置类型

Python的内置类型按照类来分，包括了 numerics, sequences, mappings, files, classes, instances 和 exceptions。对用户自定义类型的实例，如果__nonzero__或者__len__返回是 0 或 False 会被认为是假。

## 注意浅拷贝

```py
>>> lists = [[]] * 3
>>> lists
[[], [], []]
>>> lists[0].append(3)
>>> lists
[[3], [3], [3]]
```
