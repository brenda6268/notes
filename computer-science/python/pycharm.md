## 如何添加 PYTHONPATH

如果是在当前目录内：

right-click any folder in your project
select "Mark Directory As"
select "Sources Root"

如果是其他目录：

File
Settings
Project
Project Interpreter
Cog in top right => Show all
Select the interpreter
Click Tree Button
Click +
