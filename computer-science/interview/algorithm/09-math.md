辗转相除法(欧几里得算法).原理在于 gcd(a, b) == gcd(b, a % b)

```Python
def gcd(a, b):
    if b == 0:
        return a
    return gcd(b, a % b)
```

## 二进制

- 判断一个数是不是 2 的正整数次幂：`n > 0 && (n & (n - 1)) == 0`
- 对 2 的非负整数次幂取模：`x & (mod - 1)`
- 判断符号是否相同：`(x ^ y) >= 0`
- 获取某一位：`(a >> b) & 1`
- 遍历某个集合的子集: `for (int s = u; s; s = (s - 1) & u)`
- 一的个数：`while (n) {n = n & (n-1); count++}`

### 求幂

```C++
long long binpow(long long a, long long b) {
  long long res = 1;
  while (b > 0) {
    if (b & 1) res = res * a;
    a = a * a;
    b >>= 1;
  }
  return res;
}
```
